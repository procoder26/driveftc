# FTC Drive
## Setup
1. Declare names of each piece of hardware. See control hub ports. 
2. Run the `Ball Drive`

## Control Hub Ports
| Device        | Name in code  |
|---------------|---------------|
| Left Motor    | `left_motor`  |
| Right Motor   | `right_motor` |
| Claw Servo    | `claw`        |
| Vertical Claw | `vertical_claw`|
